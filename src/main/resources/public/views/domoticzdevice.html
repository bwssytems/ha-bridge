Domoticz Device List ({{bridge.domoticzdevices.length}})

For any Domoticz Device, use the build action buttons to generate the item addition information into the ha-bridge device and this will put you into the edit screen. Then you can modify the name to anything you want that will be the keyword for the Echo or Google Home. Also, you can go back to any helper tab and click a build action button to add another item for a multi-command. After you are done in the edit tab, click the 'Add Bridge Device' to finish that selection setup. The 'Already Configured Domoticz Devices' list below will show what is already setup for your Domoticz.

Also, use this select menu for which type of dim control you would like to be generated:

Use the check boxes by the names to use the bulk addition feature. Select your items and dim control type if wanted, then click bulk add below. Your items will be added with on and off or dim and off if selected with the name of the device from the Domoticz.
